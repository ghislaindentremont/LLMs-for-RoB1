# LLMs-for-RoB1
Testing open-source LLMs on modified version of original RoBin dataset from Dias et al. (2025) (https://github.com/phdabel/robin)

RoBin_Training_Dataset: Jupyter notebook to read-in original RoBin training set and develop preprocessing (modified RoBin training set) and analysis steps for RoBin_Testing_Dataset. 
RoBin_Testing_Dataset: Jupyter notebook to read-in original RoBin testing set, conduct preprocessing (modifed RoBin testing set), evaluation, and data analysis.
